# Adalab-bda-modulo-2-evaluacion-final-MatiC-hub

Este repositorio tiene la finalidad de contener las repuestas de los dos ejercicios de la Evalucación Final del Módulo 2: "Bases de Datos y Extracción de Datos"

Comienzo por el ejercicio 2.- Base de Datos Sakila

Ejercicios DE CONSULTAS para nombrar adecuadamente el orden: SELECT - FROM - WHERE
Con querys en las que es necesario usar INNER/LEFT JOIN, GROUP BY, ORDER BY, REGEXP, LIKE o BETWEEN entre otros filtros, combinaciones y funciones.

He realizado paso por paso los ejercicios para familiarizarme y comprender las consultas en lenguaje natural y trasladarlas a código SQL.

Por otro lado, había ejercicios en los que había más de una opción por lo que, en algunos, he puesto varias opciones válidas, incluidas subconsultas.

Primeramente he planteado mi resultado y luego lo he pasado por ChatGPT para ver opciones y comprenderlo en profundidad.

Muy buen ejercicio para quitarse el miedo a las consultas y al uso de Bases de Datos.

Ejercicio 1.- Creación de una base de datos de películas extraídas de un API

Se comienza importando las distintas librerías necesarias para todo el proceso de extracción de datos, creación de base de datos e inserción de datos en la misma, todo ello realizado desde Python, siguiendo las instrucciones del ejercicio.

Seguidamente se realizan distintas consultas para obtener información a partir de esos datos, desde SQL.

Una de las consultas se refiere a datos que no estaban inicialmente en la tabla por lo que he planteado una modificación de la tabla (ALTER TABLE) desde python, y la inserción de datos correspondientes, modificando el df.

Todo un reto para pararse a pensar cómo solucionarlo. En este caso he ido preguntando a ChatGPT también para no saltarme ningún paso.
